### 문제
세계는 균형이 잘 잡혀있어야 한다. 양과 음, 빛과 어둠 그리고 왼쪽 괄호와 오른쪽 괄호처럼 말이다.

정민이의 임무는 어떤 문자열이 주어졌을 때, 괄호들의 균형이 잘 맞춰져 있는지 판단하는 프로그램을 짜는 것이다.

문자열에 포함되는 괄호는 소괄호("()") 와 대괄호("[]")로 2종류이고, 문자열이 균형을 이루는 조건은 아래와 같다.

모든 왼쪽 소괄호("(")는 오른쪽 소괄호(")")와만 짝을 이뤄야 한다.
모든 왼쪽 대괄호("[")는 오른쪽 대괄호("]")와만 짝을 이뤄야 한다.
모든 오른쪽 괄호들은 자신과 짝을 이룰 수 있는 왼쪽 괄호가 존재한다.
모든 괄호들의 짝은 1:1 매칭만 가능하다. 즉, 괄호 하나가 둘 이상의 괄호와 짝지어지지 않는다.
짝을 이루는 두 괄호가 있을 때, 그 사이에 있는 문자열도 균형이 잡혀야 한다.
정민이를 도와 문자열이 주어졌을 때 균형잡힌 문자열인지 아닌지를 판단해보자.

### 입력
각 문자열은 마지막 글자를 제외하고 영문 알파벳, 공백, 소괄호("( )"), 대괄호("[ ]")로 이루어져 있으며, 온점(".")으로 끝나고, 길이는 100글자보다 작거나 같다.

입력의 종료조건으로 맨 마지막에 온점 하나(".")가 들어온다.
### 출력
각 줄마다 해당 문자열이 균형을 이루고 있으면 "yes"를, 아니면 "no"를 출력한다.

```
# boj_4949
from collections import deque

# 입력을 한 줄씩 계속 받는다 ('.'이 나오면 종료)
while True:
    data = input()

    # 닫는 괄호 → 여는 괄호 매핑
    braket_dic = {
        ')' : '(',
        '}' : '{',
        ']' : '[',
        '>' : '<'
    }

    # 여는 괄호 집합: ( { [ <
    open_braket = set(braket_dic.values())

    # 닫는 괄호 집합: ) } ] >
    close_braket = braket_dic.keys()
    
    # 종료 조건: 입력이 "." 하나만 들어오면 반복 종료
    if data == ".":
        break

    # 여는 괄호를 저장할 스택
    stack = deque()

    # 문자열이 균형 잡혔는지 여부
    is_valid = True

    # 문자열을 문자 단위로 검사
    for s in data:
        # 여는 괄호라면 스택에 저장
        if s in open_braket:
            stack.append(s)

        # 닫는 괄호라면
        elif s in close_braket:
            # 스택이 비어 있지 않고,
            # 가장 최근 열린 괄호가 현재 닫는 괄호와 짝이 맞으면
            if stack and stack[-1] == braket_dic[s]:
                stack.pop()  # 짝 맞는 괄호 제거
            else:
                # 짝이 안 맞거나, 닫을 괄호가 없으면 실패
                is_valid = False
                break

    # 반복이 끝난 후에도 스택에 여는 괄호가 남아 있으면 실패
    if stack:
        is_valid = False

    # 결과 출력
    if is_valid:
        print("yes")
    else:
        print("no")

```
